[参考文档](https://katex.org/docs/supported.html)

基本 KaTex 首尾需要`$`包含,例如`$X_y$`表示$X_y$

常用

| 表达式                         | 示例                                    |
|:------------------------------ |:--------------------------------------- |
| `\{\}`                         | $\{\}$                                  |
| `\ge`                          | $\ge$                                   |
| `\le`                          | $\le$                                   |
| `\ne`                          | $\ne$                                   |
| `\cdots`                       | $\cdots$                                |
| `X_y`                          | $X_y$                                   |
| `X^{y}` `X^{y+1}`              | $X^{y}$                       $X^{y+1}$ | 
| `\hat{\delta}`                 | $\hat{\delta}$                          |
| `\theta`                       | $\theta$                                |
| `\varepsilon`                  | $\varepsilon$                           |
| `\Sigma`                       | $\Sigma$                                |
| `\omega`                       | $\omega$                                |
| `\lbrace \rbrace`              | $\lbrace \rbrace$                       |
| `\langle\rangle`               | $\langle\rangle$                        |
| `\vert`                        | $\vert$                                 |
| `\emptyset`                    | $\emptyset$                             |
| `\varrho`                      | $\varrho$                               |
| `\varrho`                      | $\vartheta$                             |
| `\delta`                       | $\delta$                                |
| `\in`                          | $\in$                                   |
| `\notin`                       | $\notin$                                |
| `\cap`                         | $\cap$                                  |
| `\cup`                         | $\cup$                                  |
| `\displaystyle\bigcup_{i=1}^k` | $\displaystyle\bigcup_{i=1}^k$          |
| `\subset`                      | $\subset$                               |
| `\subseteq`                    | $\subseteq$                             |
| `\supset`                      | $\supset$                               |
| `\supseteq`                    | $\supseteq$                             |
| `\varepsilon\text{-}NFA`       | $\varepsilon\text{-}NFA$                |
| `\rightarrow`                  | $\rightarrow$                           |
| `\leftarrow`                   | $\leftarrow$                            |

块状使用`$$`包含

换行

```katex
$$
\begin{align}
1 == 1 \\
2 == 2
\end{align}
$$
```

$$
\begin{align}
1 == 1 \\
2 == 2
\end{align}
$$
